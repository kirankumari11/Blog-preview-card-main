### What I learned
I worked with local font files(ttf) using CSS @font-face. Below is the syntax:

```css
@font-face {
    font-family: "Figtree";
    src: url("./assets/fonts/static/Figtree-Medium.ttf");
    font-weight: 500;
    font-style: normal;
}
```
